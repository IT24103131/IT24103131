### 🎓 Projects on this account

**[VibeLanka – AI-Powered Psychometric Tourist Vacation Planner](https://github.com/IT24103131/2026-Y2-S2-DS-14_Project)**
A system that generates personalized Sri Lanka travel plans from quiz-based personality data. Recommendations are driven by personality traits and refined over time using Thompson Sampling (reinforcement learning) as users rate their trips.

**[Group38 – Lung Cancer Risk Prediction Model](https://github.com/IT24103131/Group38_Lung_Cancer_Risk_Prediction_Model)**
A machine learning model for predicting lung cancer risk from patient health data — built as a group project.

**[Online Clothing Store – Mobile E-Commerce Application](https://github.com/IT24101992/WMT-Assignment)**
Full-stack REST API (Node.js/Express, MongoDB Atlas, JWT auth) powering a mobile shopping app. *Repository owned by a teammate — contributed as part of a group assignment.*

### 💻 Other projects (personal account – [@tdillangakoon](https://github.com/tdillangakoon))

**[Personalized Treatment Plan Recommender](https://github.com/tdillangakoon/treatment-recommender)**
Full-stack clinical decision support system with a Random Forest classifier and a Flask REST API, using Thompson Sampling to learn from doctor ratings over time.

**[Sri Lanka Economic Data Pipeline](https://github.com/tdillangakoon/sl-economic-pipeline)**
Automated ETL pipeline pulling 25 years of Sri Lanka economic indicators from the World Bank API into PostgreSQL, feeding a live Power BI dashboard.
